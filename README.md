## Das Projekt
Die vorliegende Website ist Teil der praktischen Prüfung zum Mediengestalter Digital im Sommer 2021.

Die Website kann mit dem Öffnen der Datei "index.html" in Google Chrome (genaue Version siehe Abschnitt "Browser") betrachtet werden.


## Struktur
HTML:
Die html-Dateien befinden sich im Hauptverzeichnis:
1. index.html
2. leistungen.html
3. jubilaeum.html
4. pflegetipps.html
5. kontakt.html
6. impressum.html

CSS:
/css/main.css

JS:
/js


(SASS/SCSS):
Das Projekt wurde mit SASS/SCSS umgesetzt und im Anschluss mit dem VSC-PlugIn "Live Sass Compiler" kompiliert.
Der SCSS-Code befindet sich im Ordner css/scss.


## Fremde Skripte/Frameworks
Alle Slider auf der Website wurden mit "slick" umgesetzt.
Quelle: https://kenwheeler.github.io/slick/
Pfad im Projekt: /slick

Der Slick-Slider setzt jQuery voraus:
Quelle: https://code.jquery.com/jquery-3.6.0.min.js
Pfad im Projekt: /js/jquery-3.6.0.min.js

## Responsiveness

Die Website wurde ab einer Bildschirmbreite von 375px (Mobile) bis zu 2560px (Desktop) responsiv umgesetzt.
